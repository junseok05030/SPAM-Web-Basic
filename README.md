# Javascript 공부

## 실행하는 방법

### (node.js가 필요하다.)

```javascript
console.log("Hello world!");
```

이 문장을 HTML body밑에서 script를 만들고 그 사이에 넣는다.
그 후 HTML 라이브 서버를 연 후 F12를 누르고 콘솔을 보면 Hello world가 출력이 된걸 볼 수 있다.
